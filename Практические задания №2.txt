1. Напишите функцию, которая сортирует одномерный массив 
по убыванию.

2. Напишите функцию, которая возвращает сумму всех 
положительных
 элементов массива.

3. Найдите сумму индексов минимального и максимального 
элемента массива.

4. Напишите функцию, которая выводит на экран только главную 
диагональ двумерного массива.

5. Напишите функцию, которая присваивает 0 всем элементам 
двумерного массива, которые находятся ниже главной диагонали.   

6. Напишите функцию Vvod, которая заполняет массив случайными числами.

7. Напишите функцию, которая заполняет двумерный массив "змейкой".
Пример: 1 2 3
        6 5 4 
        7 8 9

8. Напишите функцию умножения матрицы на вектор.

9. Напишите функцию, которая определяет L2 норму 
погрешности по двум заданным двумерным массивам. 

10. Напишите функцию, которая "сглаживает" значения двумерного массива.
Значение каждого элемента массива заполняется средним арифметическим соседних элементов.
Эта операция проводится до тех пор, пока все значения массива перестанут меняться.
